# Utility scripts for Teaching Assistant Grader

## Available Scripts

### download_model.py
Download and setup Ollama models for the agent.

```bash
python scripts/download_model.py
```

### setup.py
Setup the project environment and dependencies.

```bash
python scripts/setup.py
```

### test_agent.py
Quick test script for agent functionality.

```bash
python scripts/test_agent.py
```
